## Gotchas already learned (don't re-discover these)

- **CI tool installs:** install `wasm-pack` & `cargo-deny` as **prebuilt binaries via
  `cargo-binstall`**, never `cargo install` — their latest releases need rustc > our pinned 1.89
  (`rust-toolchain.toml`), so source builds fail.
- **SSH key in CI:** store base64-encoded (raw keys corrupt → `error in libcrypto`).
- **`unsafe_code = "forbid"`** at workspace level CANNOT be relaxed by a crate `#![allow]`
  (E0453) — the `wasm` crate opts out via its own `[lints.rust] unsafe_code = "allow"` in
  Cargo.toml (and re-declares the clippy gate). `core` stays forbid.
- **`glab variable`/`-R owner/repo`** defaults to gitlab.com — run from inside the repo (infers
  the akaisys host) or pass the full git URL.
- **Offsets:** core = UTF-8 bytes; DOM/textarea/EditContext = UTF-16. Inc 0 dodged this with a
  full-buffer resync; **Increment 1 must do real conversion** (plan blocker #1).
